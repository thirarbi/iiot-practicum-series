# TF4017 Industrial Internet of Things Practicum Series
Curriculum and modules are by Dr. Ir. Eko Mursito Budi, MT IPM & Faqihza Mukhlish, ST MT Ph.D

### 📌 Overview
The TF4017 practicum focuses on bridging the gap between physical industrial processes and digital intelligence. The series covers the "Sensor-to-Cloud" pipeline, emphasizing reliability, scalability, and industrial communication protocols.
Key Learning Objectives:
- Embedded systems development for industrial environments.
- Implementation of M2M (Machine-to-Machine) protocols (MQTT, Modbus, OPC UA).
- Data visualization and dashboarding for real-time monitoring.
- Edge computing and basic predictive maintenance logic.
